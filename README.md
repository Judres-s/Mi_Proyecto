🎯 Descripción
es un aplicativo web pensado para estudiantes que tienen una incapacidad médica y no pueden asistir a clases presencialmente. El sistema les permite acceder a guías de aprendizaje actividades y material de apoyo desde casa para que puedan continuar con sus estudios y evitar el atraso aca-démico y la acumulación de trabajos.
🧩 Problema
Muchos estudiantes cuando presentan una incapacidad medica no pueden asistir a clases y termi-nan perdiendo información importante actividades y explicaciones de los docentes. Esto hace que se atrasen académicamente acumulen trabajos y sientan estrés cuando regresan nuevamente a la institución.

💡 Solución Propuesta
Se propone el desarrollo de un sistema web llamado APRENZY que permita:
• ✔ Gestión de usuarios estudiantes y docentes
• ✔ Registro e inicio de sesión en la plataforma
• ✔ Acceso a guías de aprendizaje actividades y material de apoyo
• ✔ Carga de incapacidades medicas por parte de los estudiantes
• ✔ Administración de información académica y comunicados
• ✔ Panel administrativo para la gestión de usuarios y contenidos
• ✔ Seguimiento de actividades académicas durante el tiempo de incapacidad
• ✔ Mejorar la comunicación entre docentes y estudiantes
• ✔ Continuidad del proceso académico desde casa durante la incapacidad medica
